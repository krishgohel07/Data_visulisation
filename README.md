# Data_visulisation
A comprehensive collection of data visualisation techniques using Python. This repository covers everything from basic static plots to advanced interactive dashboards.

📚 Libraries Covered
Matplotlib: The foundation of plotting in Python.
Seaborn: High-level interface for attractive statistical graphics.
Plotly & Cufflinks: Interactive, web-based visualisations.
Scikit-Learn: Visualising regression models and evaluation metrics.

📈 Topics Included
Distribution Plots: Histograms, KDE plots, and Rug plots.
Categorical Data: Bar plots, Count plots, Box plots, and Violin plots.
Matrix Plots: Heatmaps and Clustermaps (ideal for correlation analysis).
Grids: FacetGrids and PairGrids for multi-plot views.
Regression Plots: Linear model plots (lmplot) and residual analysis.
Interactive Visuals: Zoomable charts and hovering tooltips with Plotly.
